# Task Description

- This is a sample test which asks for a custom YouTube FrontEnd Application which uses YouTube API:
- The Application should be responsive.
- End user would be able to search videos within a date range.
- Videos will appear in a list view.
- Videos will open in Cinematic View if played.
- User can anytime click on any page button number and can view the specific page and its content.

## List of Features to be implemented

- Search bar with Auto Complete Suggestions.
- Search bar with Infinite Scrolling of the suggestions.
- A date picker where the date range will be specified for searched videos.
- There will be an "Order By" drop down which will sort the result by Date order: Newest First or Oldest First.
- Pagination with 20 results per page.

## Constraint

- Entire application should only use vanilla JavaScript.
- No Framework or Library.
